# site-baccarat-guide

本仓库用于归档和发布多个独立 HTML 页面。页面内容均为静态 HTML 文件，可直接通过浏览器打开，或部署至任意静态托管服务。

## 项目简介

该仓库主要作为 HTML 页面集合的存储与分发点。每个页面均为独立单元，不依赖外部数据库或后端服务。仓库本身不针对特定域名或网站，仅提供页面文件的版本管理与发布功能。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（如 CSS、图片、JavaScript 等）
├── index.html      # 仓库首页或导航页面（如有）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名通常反映页面内容或用途。
- `assets/`：按需组织，结构可自由调整，建议保持清晰。
- `index.html`：可选，作为仓库的入口页面。

## 页面归档说明

- 所有页面均为静态 HTML，无服务端依赖。
- 页面间相互独立，不共享状态或数据。
- 归档目的是便于长期保存与分发，不保证页面内容的实时性或外部链接的有效性。
- 页面可能包含示例数据或占位内容，用户可根据需要自行修改。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 欢迎通过 Issue 提交建议或报告问题。
- Pull Request 需遵循以下原则：
  - 新增页面请在 `pages/` 下添加独立文件。
  - 修改已有页面请说明变更原因。
  - 避免引入外部依赖或动态资源。
- 仓库不提供任何形式的保证或支持，使用者请自行评估页面内容。

## 使用方式

1. 克隆或下载本仓库。
2. 使用浏览器直接打开 `pages/` 下的 HTML 文件。
3. 或将仓库部署至任何支持静态文件的服务器或托管服务。

本仓库内容按“原样”提供，不附带任何明示或暗示的保证。使用者可自由复制、修改、分发，但需自行承担相关责任。
